# 彫り込み対象のソリッドSTL

このフォルダには、木目レリーフを彫り込みたい**対象形状のソリッドSTL**を置きます
（例: 包丁の柄）。`samples/images/` の木目画像と組み合わせて使います。

## 置き方

```
samples/shapes/knife_handle.stl   # 例
```

## 想定する処理（面ごと2.5D彫り）

包丁の柄のような3Dソリッドでも、片面ずつ「上から見て高さが一意に決まる面
（2.5D）」として扱えば、3軸でその面に木目を彫れます。

1. 対象STLから、指定方向の**上面高さマップ `H(x,y)`** と部品マスクを抽出
2. 木目深さ `d(x,y)` をマスク内だけ差し引く → `Z = H(x,y) − d(x,y)`
3. その面の変位メッシュ（プレビュー）とラスターNCを生成
4. 裏面など別の面は、向きを変えて同様に処理（マルチセットアップ）

## 注意

- 全周（側面・裏への回り込み）を1パスで巻くのは3軸では不可。回転軸加工の領域。
- 合成ソリッドをブール演算で作るのは不安定なため、「柄ソリッドはそのまま、
  木目は表面仕上げのツールパスとして適用」と割り切る方針を推奨。
- 湾曲面では平面画像が引き伸ばされる（投影方式の選択で緩和可能）。
- 大きなSTL（数十MB級）はリポジトリを膨らませるので注意。生成物（output/）は
  従来どおりコミットしない。

> 入力STLは `.gitignore` の `*.stl` 除外対象です。このフォルダの対象STLを
> リポジトリで追跡したい場合は、コミット時に `git add -f samples/shapes/xxx.stl`
> で明示的に追加してください（生成物 output/ は引き続き除外）。
